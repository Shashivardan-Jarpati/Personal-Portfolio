# A Personal Portfolio Website
A modern, responsive personal portfolio website showcasing my skills, projects, certifications, and professional background with a clean dark-themed UI.
​

## Features
Fixed Navigation: Sticky header with links to Home, About, Skills, Projects, and Contact sections.

Hero Section: Gradient background with profile placeholder, tagline, and call-to-action button.

About Section: Developer bio, experience highlights, and resume download link with visual placeholder.

Skills Grid: Six skill cards covering Web Design, Front-End, Back-End, Responsive Design, Performance Optimization, and Problem Solving.

Projects Grid: Four project cards with gradient images for E-Commerce, Social Media App, Business Dashboard, and Learning Management System.

Contact Form: Functional form with name, email, message fields plus contact details (email, phone, location).

Footer: Social media icons and copyright notice.
​

## Sections Breakdown
Section	Key Elements	Purpose
Hero	Gradient hero, profile image, CTA button	Introduction and first impression 
​
About	Text bio, resume link, image placeholder	Professional background 
​
Skills	3x2 grid cards with icons	Technical competencies 
​
Projects	2-column cards with links	Portfolio showcase 
​
Contact	Form + details grid	Client inquiries 
​
## Usage
Save as portfolio.html and open in any modern browser.

Replace placeholder images (profile, projects) with actual assets.

Update contact details, project links, and add resume.pdf.

Customize colors via CSS variables or gradients for branding.
